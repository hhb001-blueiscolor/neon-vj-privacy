üì± 1. Introduction

What is NeoN VJ

NeoN VJ is an iOS/iPadOS app that automatically recognizes music playing at DJ events and displays song titles, artist names, and album artwork in real-time.

It supports external monitors and projectors, designed for use in VJ performances and live streaming.

System Requirements

  • OS: iOS 15.6 or later
  • Devices: iPhone / iPad
  • Internet Connection: Required
  • Microphone Access: Required (for music recognition)

Features

  • ‚úÖ Automatically recognize and display playing music
  • ‚úÖ Output to external displays and projectors
  • ‚úÖ Customize event logos and backgrounds
  • ‚úÖ Display DJ names
  • ‚úÖ Publish real-time playlists on the web
  • ‚úÖ Customize layouts (7 preset options)
  • ‚úÖ 8-language support (Japanese, English, Spanish, French, Korean, Thai, Simplified Chinese, Traditional Chinese)

Limitations

  • ‚ùå No offline music recognition
  • ‚ùå Cannot recognize songs not in Shazam database
  • ‚ùå Cannot analyze recorded audio files (real-time recognition only)

üöÄ 2. Quick Start

Initial Setup

Important steps when launching the app for the first time.

1. Launch the App

When you launch the app, a welcome tutorial will automatically appear after 3 seconds on first launch.

  • The tutorial introduces basic usage in 3 steps
  • You can skip it using the "Skip" button
  • You can view the tutorial again later from the settings screen

2. Grant Microphone Access

On first launch, or when you first press the "Start" button, you will be asked for microphone access permission.

‚ö†Ô∏è Important: Always select "Allow"
  • If you select "Don't Allow", music recognition features will not work at all
  • To grant permission later:
    1. Open the iOS "Settings" app
    2. "Privacy & Security" ‚Üí "Microphone"
    3. Turn ON "NeoN VJ"

3. Test Operation (Recommended)

We recommend using the "Mic Test" button to verify that audio input is working properly.

Start Music Recognition

  1. Tap the "Start" button
  2. Wait in an environment where music is playing
  3. When recognition succeeds, song title, artist name, and album artwork will be displayed

Stop Music Recognition

Tap the "Stop" button to stop recognition and return to the "Searching..." screen

üéÆ 3. Basic Operations

Understanding the Screen

Control Buttons (Left Side)

  • Mic Test: Check audio input operation
  • Start/Stop: Start or stop music recognition
  • Retry: Re-attempt music recognition (shown only after recognition starts)
  • Settings: Open detailed settings screen
  • Position Adjustment: Shown only when external display is connected
  • Hide Menu: Hide control buttons and logs to maximize display area
‚ö†Ô∏è Important: If showing the device screen to customers, always press the "Hide Menu" button
  • System menu and recognition logs will be hidden from customers
  • When using on the floor, you can show only the music information to customers
  • External monitor output automatically has clean display without menu UI and logs

Control Buttons (Right Side)

  • Full Screen: Switch to full-screen display mode

Status Display (Bottom)

  • üü¢ Green dot: Recognition active and song is displayed
  • üü† Orange dot: Recognition active but no song recognized
  • ‚ö´ Gray dot: Recognition stopped

How to Use Retry Feature

The Retry feature is used when song recognition is difficult or when the display doesn't switch from the previous song.

When to Use

  • When song changes are not recognized
  • When you want to continue recognition before returning to the "Searching..." screen

How to Use

  1. Tap the orange "Retry" button on the left side of the screen
  2. The timeout period is reset and enters high-speed recognition mode for 5 seconds
üí° Effective Usage
  • Hard-to-recognize songs: Press the button periodically (e.g., every 25 seconds) to maintain display
  • Song transitions: Manually trigger re-recognition when a new song is not recognized

‚öôÔ∏è 4. Recommended Settings for Beginners

Open the settings screen from the "Settings" button on the left side of the screen.

üí° Settings Menu Optimized for Beginners
  • By default, only basic settings are displayed to avoid confusion for beginners
  • Advanced specialized features are hidden
  • When you're ready for more detailed customization, refer to the "8. Advanced Features" section

Change Layout

You can select the display position of music information from 7 presets.

  1. Find the "Layout" section in settings screen
  2. Tap "Preset Selection"
  3. Select your preferred layout

Layout Presets

  • Center: Album artwork and music info centered (default)
  • Minimal: Simple and compact display
  • Sidebar Left: Music info aligned to left
  • Sidebar Right: Music info aligned to right
  • Bottom Left: Album artwork at bottom (left-aligned)
  • Bottom Center: Album artwork at bottom (centered)
  • Bottom Right: Album artwork at bottom (right-aligned)
üí° When using custom backgrounds, adjusting the layout to match the background improves readability

Set Event Logo

You can display an event logo at the top right of the screen.

  1. Find the "Event Logo Settings" section in settings screen
  2. Tap "Select Logo Image"
  3. Select logo image from photo library
  4. Adjust "Logo Size" as needed
üí° If using right-aligned layout, hiding the logo prevents information overlap

Display DJ Name

You can display custom DJ names in place of "Powered by NeoN".

  1. Find the "DJ Name Display Settings" section in settings screen
  2. Turn ON "Enable DJ Name Display Mode"
  3. Enter DJ name(s) to display in the DJ name field (up to 20 names can be registered)
  4. Select the DJ name to display using radio buttons
  5. Adjust the switching interval between "Powered by NeoN" and DJ name with "Switch Interval" (3-60 seconds)

Usage Examples

  • "DJ Chromaty Yuya Playing"
  • "DJ SMYLE Playing"

Adjust Recognition Accuracy

You can adjust the balance between recognition accuracy and speed according to the situation.

  1. Find the "Music Recognition Settings" section in settings screen
  2. Select one of the following from "Recognition Preset Selection"

Recognition Presets

Stable (Recommended)
  • ‚úÖ Advantages: Prevents false recognition, stable display results
  • ‚ùå Disadvantages: Initial recognition and song transitions take more time
  • üí° Recommended for: When prioritizing display accuracy, environments with infrequent song changes
Shazam Native (Fast)
  • ‚úÖ Advantages: Recognition speed comparable to the original Shazam app
  • ‚ùå Disadvantages: May display incorrect recognition results
  • üí° Recommended for: Real-time display at live DJ events, environments with frequent song changes
Custom
  • ‚úÖ Advantages: Can adjust detailed settings individually
  • ‚ùå Disadvantages: Incorrect adjustments may prevent proper song recognition
  • üí° Recommended for: Advanced users, fine-tuning required for special environments
‚ö†Ô∏è "Custom" is for advanced users. We recommend "Stable" or "Shazam Native" for basic use.

üì∫ 5. Outputting to External Display

NeoN VJ automatically detects external display connections and starts dedicated display output.

Wired Connection Method (Recommended)

  1. Connect to projector or monitor using iPad HDMI adapter
  2. External screen display starts automatically upon cable connection
  3. iPad becomes control device, external screen becomes display-only automatically
  4. When connection succeeds, bottom log shows "Started display on external screen"
üí° AirPlay is available but may cause delays, so wired connection is recommended for important events

How to Use Position Adjustment

When video is slightly cut off on projector or external monitor, you can fine-tune the overall display position.

Operation Method

  1. Launch the app with external display connected
  2. Tap "Position Adjustment" button in left control panel (turns green)
  3. A cross-key-like adjustment panel appears
  4. Fine-tune video position with each button (moves 5% of screen per tap)
    • ‚Üë (Up arrow): Move entire video up
    • ‚Üì (Down arrow): Move entire video down
    • ‚Üê (Left arrow): Move entire video left
    • ‚Üí (Right arrow): Move entire video right
    • ‚óè (Center button): Reset position to initial state (center)
  5. Tap green "Done" button to exit adjustment mode
‚ö†Ô∏è Note: Position adjustments are not saved. Position resets to center when app restarts.

üåü 6. Useful Features

Real-time Playlist Publishing

Publish recognized songs on a dedicated web page and share the URL on social media.

Features

  • ‚úÖ Automatically publish history to web simultaneously with song recognition
  • ‚úÖ Updates every 60 seconds
  • ‚úÖ Check history in real-time from smartphone browser
  • ‚úÖ Prevent song overlap with other DJs during events
  • ‚úÖ Accumulate performance history data for each event

Setup Method

  1. Find the "Real-time Playlist Publishing" section in settings screen
  2. Turn ON "Publishing Settings"
  3. Enter "Event Name"
  4. Tap "Configure" button to get publishing URL
  5. Copy URL and share on social media
üí° DJ names set in DJ Name Display Settings will also appear on the web page

Notes

‚ö†Ô∏è About Unrecognized Songs
Songs that NeoN VJ cannot recognize will not appear in the playlist
‚ö†Ô∏è DJ Transition Notes
DJ name display may reflect unintended content during DJ transitions

Recommended procedure (during DJ transitions):
  1. Press "Stop" button on main screen to pause song recognition
  2. Change DJ name to the next DJ
  3. Press "Start" button when next DJ selects first track
This procedure prevents incorrect entries in the playlist.

  3. Turn ON "Enable Custom Background"

  • Hue: Background hue

Set auto-switch time for videos in 5-60 second range with "Switch Interval" (only for sequential/random play)

Capacity Limits

  • Maximum 400MB total for all videos

üÜò 7. Troubleshooting

When Songs Are Not Recognized

Checklist

  1. ‚úÖ Is the device connected to the Internet
  2. ‚úÖ Check audio input with Mic Test button
  3. ‚úÖ Is music volume sufficient
  4. ‚úÖ Is ambient noise too loud
  5. ‚úÖ Verify microphone access permission (Settings ‚Üí Privacy ‚Üí Microphone)

Solutions

  • Check recognition stabilization settings: Improve recognition speed with "Shazam Native" mode
  • Use "Retry" button: Manually re-attempt recognition during recognition

When Microphone Access is Denied

A "Microphone Access Permission" alert appears on screen.

  1. Tap "Open Settings" button
  2. Turn ON microphone access for "NeoN VJ" in iOS settings screen
  3. Return to app and retry "Start"

When Display is Hard to Read

  • Switch between white/black text in text color settings
  • Adjust brightness/contrast when using custom backgrounds
  • Adjust font size appropriately
  • Improve readability by adjusting Custom Info image size

When External Display Shows Nothing

  1. Check cable connection
  2. Check external display input selection
  3. Verify iPad recognizes external display
  4. Check for "Started display on external screen" message in log display

When External Display Video is Cut Off

  1. Use Position Adjustment feature: Tap "Position Adjustment" button on left side
  2. Fine-tune video position with cross keys (up to 20% in all directions)
  3. Reset position with center button
  4. Tap "Done" button after adjustment

When Videos Don't Play (iPad Only)

  1. Verify video file is supported format (MP4, MOV, M4V)
  2. Verify file size is within limit (100MB or less)
  3. Verify video slot is "Active"
  4. For single play mode: Verify video is selected with radio button
  5. Verify operation on iPad (video feature not available on iPhone)

When App Crashes

  1. Restart device
  2. Completely quit app and restart
  3. Check sufficient storage space in iOS settings
  4. Update app to latest version
  5. If using many videos: Reduce number of videos to lighten load

üéì 8. Advanced Features

üîì Show All Settings Menu

To use advanced features, turn ON "Show All Settings Menu" button in settings screen.

  • By default, only basic features are shown to avoid confusion for beginners
  • When turned ON, more specialized settings like OSC integration and detailed recognition parameter adjustments become visible
  • Use this when you're ready to customize according to your preferences

OSC Integration (VJ Software Integration)

Using OSC (Open Sound Control) feature, you can integrate with VJ software (Resolume, MadMapper, etc.).

Features

  • Send song information recognized by NeoN VJ to VJ software in real-time
  • Send song title, artist name, album name, confidence level, recognition time
  • Support for multiple destination slots

Setup Method

  1. Find the "OSC OUTPUT" section in settings screen
  2. Turn ON "Enable OSC Output"
  3. Add destination slot and set IP address and port number
  4. Select data to send
üí° This feature is highly specialized, intended for those with VJ software experience.

Operation Mode Settings

You can switch the app's operation mode.

iPhone Mode

  • Can use display sizes optimized for smaller screens like iPhone or iPad mini
  • Suitable for small screens and close-range display

Admin Mode

  • Exclusive feature for NeoN members (password authentication required)
  • General users don't need to use this

Normal Display Settings

You can fine-tune the size of each display element.

Adjustable Items

  • Album Artwork Size: Adjust in range 120-800
  • Song Title Font Size: Adjust in range 50-150
  • Artist Name Font Size: Adjust in range 30-120
  • Logo Size: Adjust in range 30-200
  • Credit Font Size: Adjust in range 10-50
  • Custom Info Image Size: Adjust in range 50-500
üí° Generally, the layout preset feature introduced in "4. Recommended Settings for Beginners" is sufficient, but use this for more detailed size adjustments

Large Text Display Mode

Text-only large display mode.

  • When "Enable Large Text Display Mode" is ON, album artwork is hidden for text-only large display
  • More visible from distance
  • Suitable for large floors or spacious venues

Video Background (iPad Only)

‚ö†Ô∏è This is an advanced feature. For basic setup, refer to the "6. Useful Features" section

Video background is an advanced iPad-only feature. Displaying videos as background enables more spectacular effects.

External Display Settings

Access advanced external display settings.

Settings Items

  1. Find the "External Display" section in settings screen (need to turn ON "Show All Settings Menu")
  2. Toggle "Auto External Display" ON/OFF
  3. Check connection status (connected external display info is shown)
üí° Position Adjustment feature is available from control buttons on left side when external display is connected (refer to "5. Outputting to External Display" section)

Detailed Recognition Parameter Adjustments

When "Custom" preset is selected, you can adjust the following items individually.

  • History Retention Period: Set 5-60 second range for referencing past recognition results
  • Judgment Threshold: Set 10-100% range for confidence level to confirm songs
  • Minimum Recognition Count Before Judgment: Set 1-10 range
  • Instant Change Count for Continuous Recognition: Set 2-5 range for response speed when same song is continuously recognized
  • Silence Timeout: Set 10-120 second range for time until returning to "Searching..." when no music is recognized
‚ö†Ô∏è Incorrect adjustments may prevent proper song recognition. We recommend using "Stable" or "Shazam Native" for basic use.

üîö 9. How to Exit the App

Normal Exit Procedure

  1. Stop Music Recognition: Tap "Stop" button to end recognition
  2. Confirm Settings Saved: If settings were changed, always save with "Save" button
  3. Home Button/Gesture: Exit app normally

When Exiting After Extended Use

  • When using video backgrounds or many effects, the device may be under load
  • We recommend letting the device rest after exiting the app

When Force Quit is Necessary

If the app becomes unresponsive:

  1. Open app switcher (double-press home button, or swipe up from home bar)
  2. Swipe NeoN VJ app upward to force quit
  3. Recommend restarting iPad
