구매 전 필독

음악 인식 제한 사항

이 앱은 Apple ShazamKit을 사용하여 음악을 인식합니다. 다음 상황에서는 인식 정확도가 낮아질 수 있습니다:

  • BPM이 ±5% 이상 변경된 트랙 - 큰 피치 시프트나 템포 변경은 인식을 어렵게 합니다
  • 두 트랙을 믹싱하는 동안 - 트랜지션 중에는 인식 정확도가 낮아집니다
  • 리믹스 또는 에디트 버전 - 오리지널 버전과 다른 경우 인식되지 않을 수 있습니다

DJ 스타일에 따라 기대한 대로 인식되지 않을 수 있음을 양해해 주세요.

이것은 개발자 테스트에서 관찰된 경향이며, ShazamKit의 공식 사양이 아닙니다.

1. 소개

NeoN VJ란

NeoN VJ는 DJ 이벤트에서 재생되는 음악을 자동으로 인식하고, 곡명, 아티스트명, 앨범 아트를 실시간으로 표시하는 iOS/iPadOS 앱입니다.

외부 모니터나 프로젝터에도 대응하며, VJ 퍼포먼스나 방송에서의 활용을 목적으로 하고 있습니다.

작동 환경

  • 지원 OS: iOS 15.6 이상
  • 지원 기기: iPhone / iPad
  • 인터넷 연결: 필수
  • 마이크 액세스: 필수 (음악 인식을 위해)

가능한 기능

  • ✅ 재생 중인 음악을 자동으로 인식하여 표시
  • ✅ 외부 디스플레이 및 프로젝터로 출력
  • NDI 출력: 곡 정보를 VJ 소프트웨어와 OBS에 무선으로 전송 (투명 레이어 지원)
  • ✅ 이벤트 로고 및 배경 커스터마이징
  • ✅ DJ 이름 표시
  • ✅ 레이아웃 커스터마이징 (7가지 프리셋)
  • ✅ 8개 언어 지원 (일본어, 영어, 스페인어, 프랑스어, 한국어, 태국어, 중국어 간체, 중국어 번체)

불가능한 기능

  • ❌ 오프라인 곡 인식
  • ❌ Shazam 데이터베이스에 없는 곡 인식
  • ❌ 녹음된 오디오 파일 분석 (실시간 인식만 가능)

2. 빠른 시작

초기 설정

앱을 처음 실행할 때의 중요한 단계입니다.

1. 앱 실행하기

앱을 실행하면, 처음 실행 시에는 3초 후에 환영 튜토리얼이 자동으로 표시됩니다.

  • 튜토리얼은 3단계로 기본 사용법을 소개합니다
  • "건너뛰기" 버튼으로 건너뛸 수 있습니다
  • 나중에 설정 화면에서 다시 튜토리얼을 볼 수 있습니다

2. 마이크 액세스 허용하기

처음 실행할 때 또는 "인식 시작" 버튼을 처음 누를 때 마이크 액세스 권한을 요청합니다.

중요: 반드시 "허용"을 선택하세요
  • "허용 안 함"을 선택하면 곡 인식 기능을 전혀 사용할 수 없습니다
  • 나중에 허용하려면:
    1. iOS "설정" 앱 열기
    2. "개인정보 보호 및 보안" → "마이크"
    3. "NeoN VJ"를 ON으로 설정

3. 작동 확인 (권장)

"마이크 테스트" 버튼으로 오디오 입력이 정상적으로 작동하는지 확인하는 것을 권장합니다.

곡 인식 시작하기

  1. "인식 시작" 버튼을 탭합니다
  2. 음악이 재생되는 환경에서 대기합니다
  3. 인식이 성공하면 곡명, 아티스트명, 앨범 아트가 표시됩니다

곡 인식 중지하기

"중지" 버튼을 탭하면 인식이 중지되고 "Searching..." 화면으로 돌아갑니다

3. 기본 조작

화면 보기

컨트롤 버튼 (화면 왼쪽)

  • 마이크 테스트: 오디오 입력 작동 확인
  • 인식 시작/중지: 곡 인식 시작 및 중지
  • 재판정: 곡 인식 다시 시도 (인식 시작 후에만 표시)
  • 설정: 상세 설정 화면 열기
  • 위치 조정: 외부 디스플레이 연결 시에만 표시
  • 메뉴 숨기기: 컨트롤 버튼과 로그 표시를 숨겨 표시 영역을 최대화
중요: 기기 화면을 고객에게 보여줄 경우 반드시 "메뉴 숨기기" 버튼을 누르세요
  • 시스템 메뉴나 인식 로그 등 불필요한 정보가 고객에게 보이지 않습니다
  • 플로어에서 사용할 때 고객에게 곡 정보만 보여줄 수 있습니다
  • 외부 모니터 출력에는 처음부터 메뉴 UI와 로그가 표시되지 않습니다 (자동으로 깔끔한 표시가 됩니다)

컨트롤 버튼 (화면 오른쪽)

  • 전체 화면 표시: 전체 화면 표시 모드로 전환

상태 표시 (화면 하단)

  • 🟢 녹색 점: 곡 인식 중이며 곡이 표시되는 상태
  • 🟠 주황색 점: 곡 인식 중이지만 인식하지 못한 상태
  • ⚫ 회색 점: 곡 인식이 중지된 상태

재판정 기능 사용법

재판정 기능은 곡 인식이 어려운 경우나 이전 곡에서 전환되지 않을 때 사용합니다.

사용 상황

  • 곡 전환이 인식되지 않을 때
  • "Searching..." 화면으로 돌아가기 전에 인식을 계속하고 싶을 때

사용 방법

  1. 화면 왼쪽의 주황색 "재판정" 버튼을 탭합니다
  2. 타임아웃 시간이 리셋되고 5초간 고속 인식 모드가 됩니다
효과적인 사용법
  • 인식 어려운 곡: 25초마다 등 정기적으로 버튼을 눌러 표시를 유지
  • 곡 전환 시: 새로운 곡이 인식되지 않을 경우 수동으로 재인식 촉진

4. 초보자를 위한 추천 설정

설정 화면은 화면 왼쪽의 "설정" 버튼에서 열 수 있습니다.

초보자를 위해 최적화된 설정 메뉴
  • 기본적으로 초보자가 헷갈리지 않도록 기본 설정 항목만 표시됩니다
  • 고급 사용자를 위한 전문 기능은 숨겨져 있습니다
  • 익숙해지고 더 자세한 커스터마이징을 원할 경우 "9. 고급 사용자 기능" 장을 참조하세요

레이아웃 변경하기

곡 정보 표시 위치를 7가지 프리셋에서 선택할 수 있습니다.

  1. 설정 화면에서 "레이아웃" 섹션을 찾습니다
  2. "프리셋 선택"을 탭합니다
  3. 원하는 레이아웃을 선택합니다

레이아웃 프리셋 목록

  • 중앙: 앨범 아트와 곡 정보를 중앙에 배치 (기본값)
  • 미니멀: 심플하고 작은 표시
  • 왼쪽 정렬: 곡 정보를 왼쪽에 배치
  • 오른쪽 정렬: 곡 정보를 오른쪽에 배치
  • 중앙·이미지 하단: 앨범 아트를 하단에 배치 (중앙)
  • 왼쪽·이미지 하단: 앨범 아트를 하단에 배치 (왼쪽 정렬)
  • 오른쪽·이미지 하단: 앨범 아트를 하단에 배치 (오른쪽 정렬)
커스텀 배경을 설정하는 경우 배경에 맞춰 레이아웃을 조정하면 보기 쉽습니다

이벤트 로고 설정하기

화면 오른쪽 상단에 이벤트 로고를 표시할 수 있습니다.

  1. 설정 화면에서 "이벤트 로고 설정" 섹션을 찾습니다
  2. "로고 이미지 선택"을 탭합니다
  3. 사진 라이브러리에서 로고 이미지를 선택합니다
  4. 필요에 따라 "로고 크기"를 조정합니다
레이아웃에서 오른쪽 정렬을 선택한 경우 로고를 숨기면 정보 겹침을 방지할 수 있습니다

DJ 이름 표시하기

"Powered by NeoN" 부분에 임의의 DJ 이름을 표시할 수 있습니다.

  1. 설정 화면에서 "DJ명 표시 설정" 섹션을 찾습니다
  2. "DJ명 표시 모드를 활성화"를 ON으로 설정합니다
  3. DJ명 항목에 표시할 DJ 이름을 입력합니다 (최대 20명까지 등록 가능)
  4. 표시할 DJ 이름을 라디오 버튼으로 선택합니다
  5. "전환 초"로 "Powered by NeoN"과 DJ 이름의 전환 간격을 조정합니다 (3~60초)

사용 예

  • "DJ 크로마티 유야 Playing"
  • "DJ SMYLE Playing"

곡 인식 정확도 조정하기

현장 상황에 따라 인식 정확도와 속도의 균형을 조정할 수 있습니다.

  1. 설정 화면에서 "곡 인식 설정" 섹션을 찾습니다
  2. "인식 프리셋 선택"에서 다음 중 하나를 선택합니다

인식 프리셋

안정성 우선 (권장)
  • ✅ 장점: 오인식을 방지하고 표시 결과가 안정적임
  • ❌ 단점: 첫 인식과 곡 전환에 시간이 걸림
  • 💡 권장 용도: 표시 정확성을 최우선으로 할 경우, 곡 변경 빈도가 적은 환경
Shazam 네이티브 (고속 우선)
  • ✅ 장점: 본가 Shazam에 버금가는 인식 속도
  • ❌ 단점: 잘못된 인식 결과가 표시될 수 있음
  • 💡 권장 용도: 라이브 DJ 이벤트의 실시간 표시, 곡 전환이 빈번한 환경
커스텀
  • ✅ 장점: 상세 설정을 개별적으로 조정 가능
  • ❌ 단점: 조정을 잘못하면 곡 인식이 정상적으로 작동하지 않음
  • 💡 권장 용도: 고급 사용자용, 특수한 환경에 맞춘 세밀한 조정이 필요한 경우
"커스텀"은 고급 사용자용입니다. 기본적으로는 "안정성 우선" 또는 "Shazam 네이티브"를 권장합니다.

5. 외부 디스플레이로 출력하기

NeoN VJ는 외부 디스플레이 연결을 자동으로 감지하고 전용 표시를 시작합니다.

유선 연결 방법 (권장)

  1. iPad용 HDMI 어댑터로 프로젝터나 모니터에 연결합니다
  2. 케이블 연결과 동시에 자동으로 외부 화면 표시가 시작됩니다
  3. iPad 본체는 조작용, 외부 화면은 표시 전용으로 자동 작동 시작
  4. 연결 성공 시 화면 하단 로그에 "외부 디스플레이에 표시를 시작했습니다"라고 표시됩니다
AirPlay도 사용할 수 있지만 지연이 발생할 수 있으므로 중요한 이벤트에서는 유선 연결을 권장합니다

위치 조정 기능 사용법

프로젝터나 외부 모니터에서 영상이 약간 잘리는 경우 영상 전체의 표시 위치를 미세 조정할 수 있습니다.

조작 방법

  1. 외부 디스플레이를 연결한 상태에서 앱을 실행합니다
  2. 화면 왼쪽 컨트롤 패널에서 "위치 조정" 버튼을 탭합니다 (녹색으로 변함)
  3. 십자 키 같은 조정 패널이 표시됩니다
  4. 각 버튼으로 영상 위치를 미세 조정 (1회 탭으로 화면의 5%씩 이동)
    • ↑ (위쪽 화살표): 영상 전체를 위로 이동
    • ↓ (아래쪽 화살표): 영상 전체를 아래로 이동
    • ← (왼쪽 화살표): 영상 전체를 왼쪽으로 이동
    • → (오른쪽 화살표): 영상 전체를 오른쪽으로 이동
    • ● (중앙 버튼): 위치를 초기 상태 (중앙)로 리셋
  5. 녹색 "완료" 버튼을 탭하여 조정 모드를 종료합니다
주의: 위치 조정은 저장되지 않습니다. 앱 재시작 시 중앙 위치로 리셋됩니다.

6. Source 모드 (NDI / HDMI 출력)

Source 모드는 곡 정보를 외부 VJ 소프트웨어(OBS, Resolume 등)로 전송하는 기능입니다. NDI 프로토콜을 사용하여 동일 네트워크상의 PC나 Mac에서 직접 수신할 수 있습니다.

NDI란?

NDI(Network Device Interface)는 네트워크를 통해 실시간으로 영상 및 오디오를 전송하는 업계 표준 프로토콜입니다. OBS, Resolume, vMix, Wirecast 등 주요 VJ/스트리밍 소프트웨어에서 수신할 수 있습니다.

Display 모드와 Source 모드의 차이

NeoN VJ에는 두 가지 동작 모드가 있습니다.

  • Display 모드(기본값): iPad 본체나 외부 디스플레이에 배경, 로고, 이펙트가 포함된 완성된 화면을 표시하는 모드입니다. 일반적인 사용에서는 이 모드를 사용합니다
  • Source 모드: 곡 정보(곡명, 아티스트명, 앨범 아트)를 NDI를 통해 외부로 전송하는 모드입니다. 수신 측의 VJ 소프트웨어에서 자유롭게 합성할 수 있습니다. HDMI 연결 시에도 Source 모드 화면이 표시되지만, 배경은 검은색으로 표시됩니다
모드 선택 기준
  • iPad 단독 또는 외부 모니터 직결 사용 → Display 모드
  • OBS나 Resolume 등 VJ 소프트웨어에 곡 정보를 보내고 싶을 때 → Source 모드 (NDI)

외부 출력 동작 비교

Display 모드와 Source 모드는 외부 출력 동작이 다릅니다.

Display 모드 + HDMI

  • HDMI로 연결된 외부 디스플레이에 배경, 로고, 이펙트가 포함된 완성된 화면이 그대로 표시됩니다
  • iPad 본체는 조작용, 외부 화면은 표시 전용으로 동작합니다
  • 프로젝터나 모니터에 직접 투영하는 용도에 최적입니다

Source 모드 + NDI (권장)

  • 곡 정보가 투명 레이어(알파 채널 포함)로 네트워크를 통해 전송됩니다
  • 수신 측의 VJ 소프트웨어에서 자신의 영상 위에 곡 정보만 오버레이 합성할 수 있습니다
  • 배경이 투명하므로 곡 정보 텍스트와 앨범 아트만 표시됩니다

Source 모드 + HDMI

  • Source 모드 활성 상태에서 HDMI를 연결하면 외부 디스플레이에도 Source 모드 화면이 표시됩니다
  • 단, HDMI는 투명 레이어를 지원하지 않으므로 배경은 검은색으로 표시됩니다
  • 검은 배경 위에 곡 정보가 표시되는 형태가 됩니다
투명 합성을 원하시면 NDI를 사용하세요. HDMI 연결에서는 알파 채널(투명 정보)을 전송할 수 없으므로 수신 측에서의 투명 합성이 불가능합니다.

Source 모드 활성화

  1. 설정 화면을 엽니다
  2. "동작 모드" 섹션에서 "Source Mode (NDI/HDMI) 활성화 [Beta]"를 켭니다
  3. 화면이 Source 모드 전용 표시로 전환되고 NDI 전송이 자동으로 시작됩니다
주의: Source 모드를 활성화하면 일반 Display 모드 화면에서 전환됩니다. 되돌리려면 Source 모드 화면의 "Display" 버튼을 탭하세요.

Source 모드 화면 구성

미리보기 영역

화면 중앙에 NDI로 전송되는 영상의 미리보기가 표시됩니다. 투명 레이어 위에 곡 정보가 렌더링되는 모습을 실시간으로 확인할 수 있습니다.

상태 표시줄

화면 상단에 전송 상태가 실시간으로 표시됩니다.

  • NDI(녹색 점): NDI 전송 중
  • fps: 프레임 레이트
  • ms: 캡처 지연
  • 수신기 아이콘 + 숫자: 연결된 NDI 수신기 수
  • 해상도: 출력 해상도 (1920x1080)

조작 버튼

  • Display: Display 모드로 돌아가기
  • Start / Stop: 음악 인식 시작/중지
  • Hold Display: 현재 표시 중인 곡 정보 고정 (곡이 바뀌어도 표시 유지)
  • Retry: 음악 인식 재시도
  • Blackout: NDI 연결을 유지한 채 영상 출력을 일시적으로 숨기기
  • Settings: 설정 화면 열기

출력 요소 선택

NDI로 전송하는 요소를 개별적으로 ON/OFF할 수 있습니다.

  • Album Art: 앨범 아트워크
  • Song Title: 곡명
  • Artist: 아티스트명
  • Powered by / DJ: 크레딧 표시 / DJ명
  • Searching Text: 인식 중 "Searching..." 텍스트
예를 들어, VJ 소프트웨어 측에서 자체 앨범 아트를 표시하고 있다면, NeoN VJ 측의 Album Art를 OFF로 하고 곡명과 아티스트명만 전송할 수 있습니다.

레이아웃 프리셋 (9종)

Source 모드 전용 레이아웃 프리셋으로 출력 요소의 배치 위치를 선택할 수 있습니다.

  • Center: 중앙 배치 (기본값)
  • Center Top: 중앙 상단
  • Center Bottom: 중앙 하단
  • Center Left: 좌측 중앙
  • Center Right: 우측 중앙
  • Top Left: 좌측 상단
  • Top Right: 우측 상단
  • Bottom Left: 좌측 하단
  • Bottom Right: 우측 하단
VJ 소프트웨어의 영상 레이아웃에 맞춰 곡 정보 배치 위치를 선택하세요. 예를 들어, VJ 영상 하단에 자막처럼 곡 정보를 표시하려면 "Center Bottom"이 적합합니다.

NDI 수신 측 설정

필요 사항

  • iPad와 수신 PC/Mac이 동일 네트워크에 연결되어 있어야 합니다
  • NDI 호환 소프트웨어 (OBS + NDI 플러그인, Resolume, vMix 등)

OBS 설정 예시

  1. OBS NDI 플러그인을 설치합니다
  2. OBS에서 "소스" → "NDI Source"를 추가합니다
  3. 소스 이름 목록에서 "NeoN VJ"를 선택합니다
  4. 영상이 투명 레이어로 수신되어 다른 소스 위에 합성할 수 있습니다
투명 합성에 대한 자세한 내용은 위의 "외부 출력 동작 비교" 섹션을 참조하세요.

Source 모드 문제 해결

NDI 수신 측에서 소스를 찾을 수 없는 경우

  1. iPad와 수신 기기가 동일 Wi-Fi 네트워크에 연결되어 있는지 확인
  2. Source 모드가 활성화되어 있는지 확인 (상태 표시줄에 녹색 "NDI" 표시)
  3. 방화벽이 NDI 통신을 차단하고 있지 않은지 확인
  4. NDI 수신 소프트웨어의 소스 목록을 새로고침

영상이 끊기거나 지연되는 경우

  • Wi-Fi 신호 강도 확인 (5GHz 대역 권장)
  • 네트워크상의 다른 트래픽 줄이기
  • 상태 표시줄의 "drop"(드롭 프레임 수)이 증가하고 있지 않은지 확인

7. 유용한 기능

커스텀 배경 및 필터 설정

이벤트용 배경 이미지를 설정하고 필터로 조정할 수 있습니다.

배경 이미지 설정

  1. 설정 화면에서 "커스텀 배경 설정" 섹션을 찾습니다
  2. "배경 이미지 선택"으로 이벤트용 배경 이미지를 설정합니다
  3. "커스텀 배경 활성화"를 ON으로 설정합니다

필터 조정

문자 가독성을 조정할 수 있습니다.

  • 밝기: 배경 밝기
  • 대비: 배경 대비
  • 채도: 배경 색의 선명도
  • 블러: 배경 흐림 정도
  • 색조: 배경 색상
  • 투명도: 배경 투명도
배경 이미지를 설정한 경우 레이아웃 프리셋으로 배치를 조정하면 보기 쉽습니다

동영상 배경 (iPad 전용)

이 기능은 iPad에서만 사용할 수 있습니다

배경에 동영상을 표시할 수 있습니다.

동영상 업로드

  1. 설정 화면에서 "커스텀 동영상 설정" 섹션을 찾습니다
  2. "동영상 배경 활성화"를 ON으로 설정합니다
  3. 슬롯 1~5에 동영상 파일을 업로드합니다 (각 슬롯의 "추가" 버튼을 탭)
  4. 지원 형식: MP4, MOV, M4V
  5. 각 슬롯의 ON/OFF 전환: 업로드 후 각 동영상의 활성/비활성을 개별적으로 설정 가능

재생 모드

  • 단일 재생: 선택한 1개의 동영상을 반복 재생 (라디오 버튼으로 동영상 선택)
  • 순차 재생: 활성 동영상을 순서대로 자동 전환
  • 랜덤 재생: 활성 동영상을 무작위 순서로 자동 전환

전환 간격 설정

"전환 간격"으로 5~60초 범위에서 동영상 자동 전환 시간을 설정합니다 (순차 재생 및 랜덤 재생 시에만 유효)

용량 제한

  • 파일당 최대 100MB
  • 전체 동영상 합계 최대 400MB
  • 권장 해상도: Full HD (1920×1080) 이하
  • 지원 파일 형식: MP4, MOV, M4V

YouTube 동영상 배경 (MV 표시·β)

※이 기능은 현재 베타 버전으로 제공됩니다. 동영상이 재생되지 않거나 다른 동영상이 선택되는 경우가 있습니다.

인식한 곡의 뮤직비디오(MV)를 YouTube에서 자동으로 찾아 배경에 표시하는 기능입니다. 곡에 따라서는 MV를 찾지 못해 재생되지 않는 경우가 있습니다.

필요한 것: 인터넷 연결과 YouTube API 키(무료). 키를 얻는 방법은 「YouTube API 키 발급 가이드」에서 처음 하시는 분을 위해 하나씩 설명합니다.

사용법: 설정 → 「모든 설정 메뉴 표시」를 ON → 「YouTube 동영상 배경」에 API 키를 붙여넣기 → 「저장 후 확인」 → ✅가 나오면 기능을 ON.

동영상 화질: 보통은 「자동」 그대로 두면 됩니다. 행사장의 통신이 불안정해 영상이 자주 멈출 때는 720p나 480p로 낮추면 재생이 안정되기 쉽습니다.

아울러 「MV 재생 중 앨범 아트 숨기기」도 선택할 수 있습니다. 하루 검색 횟수에는 무료 한도의 상한이 있으며, 설정 화면의 사용량 미터에서 확인할 수 있습니다(상한 도달 시 다음 날 자동 초기화).

언어 설정

앱 표시 언어를 변경할 수 있습니다.

  1. 설정 화면에서 "언어 설정" 섹션을 찾습니다
  2. 원하는 언어를 선택합니다

지원 언어

일본어, 영어, 스페인어, 프랑스어, 한국어, 태국어, 중국어 간체, 중국어 번체

8. 문제 해결 (트러블슈팅)

곡이 인식되지 않는 경우

체크리스트

  1. ✅ 기기가 인터넷에 연결되어 있는지 확인
  2. ✅ 마이크 테스트 버튼으로 오디오 입력 확인
  3. ✅ 음악 볼륨이 충분한지 확인
  4. ✅ 주변 소음이 너무 크지 않은지 확인
  5. ✅ 마이크 액세스 권한 확인 (설정 → 개인정보 보호 → 마이크)

대처 방법

  • 인식 안정화 설정 확인: "Shazam 네이티브" 모드로 인식 속도 향상
  • "재판정" 버튼 사용: 인식 중 수동으로 인식 재시도

마이크 액세스가 거부된 경우

화면에 "마이크 액세스 권한" 알림이 표시됩니다.

  1. "설정 열기" 버튼을 탭합니다
  2. iOS 설정 화면에서 "NeoN VJ"의 마이크 액세스를 ON으로 설정합니다
  3. 앱으로 돌아가서 "인식 시작"을 다시 시도합니다

표시가 보기 어려운 경우

  • 문자 색상 설정에서 흰색 문자, 검은색 문자를 전환
  • 커스텀 배경 사용 시 밝기 및 대비를 조정
  • 폰트 크기를 적절히 조정
  • 커스텀 Info 이미지 크기를 조정하여 가독성 개선

외부 디스플레이에 표시되지 않는 경우

  1. 케이블 연결 확인
  2. 외부 디스플레이의 입력 전환 확인
  3. iPad에서 외부 디스플레이가 인식되고 있는지 확인
  4. 로그 표시에서 "외부 디스플레이에 표시를 시작했습니다" 메시지 확인

외부 디스플레이 영상이 잘리는 경우

  1. 위치 조정 기능 사용: 화면 왼쪽의 "위치 조정" 버튼을 탭
  2. 십자 키로 영상 위치를 미세 조정 (상하좌우로 최대 20%까지)
  3. 중앙 버튼으로 위치 리셋
  4. 조정 완료 후 "완료" 버튼으로 종료

동영상이 재생되지 않는 경우 (iPad 전용)

  1. 동영상 파일이 지원 형식 (MP4, MOV, M4V)인지 확인
  2. 파일 크기가 제한 내 (100MB 이하)인지 확인
  3. 동영상 슬롯이 "활성"인지 확인
  4. 단일 재생 모드의 경우: 라디오 버튼으로 동영상이 선택되어 있는지 확인
  5. iPad에서 작동 확인 (iPhone에서는 동영상 기능 사용 불가)

앱이 충돌하는 경우

  1. 기기를 재시작합니다
  2. 앱을 완전히 종료한 후 재시작합니다
  3. iOS 설정에서 충분한 저장 공간이 있는지 확인합니다
  4. 앱을 최신 버전으로 업데이트합니다
  5. 동영상을 많이 사용하는 경우: 동영상 수를 줄여 부하를 경감합니다

YouTube 동영상 배경이 잘 작동하지 않는 경우

API 키의 「저장 후 확인」 결과별 대처는 YouTube API 키 발급 가이드의 「잘 안될 때」를 참고하세요.

곡에 따라 MV가 표시되지 않는 것은 이상이 아닙니다(공식 MV를 찾지 못하는 경우 등). 많은 곡에서 표시되지 않는 경우에는 사용량 미터가 상한에 도달하지 않았는지 확인하세요.

9. 고급 사용자 기능

모든 설정 메뉴 표시하기

고급 사용자 기능을 이용하려면 설정 화면에서 "설정 메뉴 모두 표시" 버튼을 ON으로 설정하세요.

  • 기본적으로 초보자가 헷갈리지 않도록 기본 기능만 표시됩니다
  • ON으로 설정하면 상세한 인식 파라미터 조정 등 더욱 전문적인 설정 항목이 표시됩니다
  • 익숙해지고 나만의 커스터마이징을 원할 경우 이용하세요

동작 모드 설정

앱의 동작 모드를 전환할 수 있습니다.

iPhone 모드

  • iPhone이나 iPad mini 등 작은 화면용 표시 크기로 사용할 수 있습니다
  • 작은 화면이나 근거리 표시에 적합합니다

일반 표시 설정

각 표시 요소의 크기를 세밀하게 조정할 수 있습니다.

조정 가능 항목

  • 앨범 아트 크기: 120~800 범위에서 조정
  • 곡명 폰트 크기: 50~150 범위에서 조정
  • 아티스트명 폰트 크기: 30~120 범위에서 조정
  • 로고 크기: 30~200 범위에서 조정
  • 크레딧 폰트 크기: 10~50 범위에서 조정
  • 커스텀 Info 이미지 크기: 50~500 범위에서 조정

대문자 표시 모드

텍스트만 큰 표시 모드입니다.

  • "대문자 표시 모드 활성화"를 ON으로 설정하면 앨범 아트를 숨기고 텍스트만 큰 표시가 됩니다
  • 더 멀리서도 보기 쉬운 표시 방식입니다
  • 큰 플로어나 넓은 회장에서 사용하기에 적합합니다

동영상 배경 (iPad 전용)

이 기능은 고급 사용자용 기능입니다. 기본 설정 방법은 "7. 유용한 기능" 장을 참조하세요

동영상 배경 기능은 iPad 전용 고급 사용자 기능입니다. 배경에 동영상을 표시하여 더욱 화려한 연출이 가능합니다.

외부 디스플레이 설정

고급 사용자용 외부 디스플레이 설정에 액세스할 수 있습니다.

설정 항목

  1. 설정 화면에서 "외부 디스플레이" 섹션을 찾습니다 ("설정 메뉴 모두 표시"를 ON으로 설정해야 합니다)
  2. "외부 디스플레이 자동 표시"의 ON/OFF 전환
  3. 연결 상태 확인 (연결된 외부 디스플레이 정보가 표시됩니다)
위치 조정 기능은 외부 디스플레이 연결 시 화면 왼쪽 컨트롤 버튼에서 이용할 수 있습니다 ("5. 외부 디스플레이로 출력하기" 장 참조)

상세한 인식 파라미터 조정

"커스텀" 프리셋을 선택하면 다음 항목을 개별적으로 조정할 수 있습니다.

  • 이력 보존 기간: 5~60초 범위에서 과거 인식 결과를 참고하는 시간 설정
  • 판정 임계값: 10~100% 범위에서 얼마나 확률로 곡을 확정할지 설정
  • 판정 시작까지 최소 인식 횟수: 1~10회 범위에서 설정
  • 연속 인식에서 즉시 변경 횟수: 2~5회 범위에서 동일 곡이 연속으로 인식되었을 때의 반응 속도 설정
  • 무음 시 타임아웃: 10~120초 범위에서 음악이 인식되지 않을 때 "Searching..."으로 돌아가는 시간 설정
조정을 잘못하면 곡 인식이 정상적으로 작동하지 않을 위험이 있습니다. 기본적으로는 "안정성 우선" 또는 "Shazam 네이티브" 사용을 권장합니다.
